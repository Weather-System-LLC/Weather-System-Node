# Weather System Node

Weather System Node is an internal tool that automatically posts daily weather forecasts and alerts. It also includes a Christmas countdown feature.

## Features

- Daily Forecasts – posts today's and tonight's weather automatically
- Alerts – monitors for new weather alerts and posts them in real-time
- Christmas Countdown – posts a countdown to Christmas each day of December

## Setup

1. Create a `.env` file in the project root with the following content and fill in your details:

```.env
FacebookToken=
CountyCode=
ForecastCode=
```
